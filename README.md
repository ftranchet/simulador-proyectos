# simulador-proyectos
Pequeño simulador para la materia evaluación de proyectos de inversión.

Permite simular hasta 5 proyectos a la vez para comparar parámetros y resultados (VAN, TIR, recupero, IVAN) entre ellos: elegí la cantidad en "Proyectos en simulación" y editá cada uno desde su pestaña.

Creado por [Francisco Tranchet](https://www.linkedin.com/in/ftranchet/) con la ayuda de la IA.
